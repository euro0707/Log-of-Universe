# 背景画像について

このフォルダには以下の画像ファイルを配置してください：

1. bg1.png 〜 bg5.png: 宇宙/銀河/星空などの背景画像
   - 推奨サイズ: 1920×1080px
   - 宇宙、銀河、星雲などの神秘的な画像

2. stars.png: 星空の背景用画像
   - 黒い背景に小さな星が散りばめられた画像
   - タイル表示可能なもの

3. twinkling.png: きらめき効果用の透過画像
   - 透明背景に白い点や光の粒子が散りばめられた画像
   - アニメーション効果用

著作権フリーの宇宙画像を使用することをお勧めします。
以下のサイトから入手できます：

- Unsplash (https://unsplash.com/s/photos/space)
- Pexels (https://www.pexels.com/search/space/)
- NASA Image Gallery (https://www.nasa.gov/multimedia/imagegallery/)
